🎬 What is CutLab?

CutLab is a full-stack web-based video editing tool built with Node.js and Express. It allows users to create video editing projects, upload video files, trim clips by setting start and end times, add notes, and manage everything through a clean and responsive interface.

🎯 Project Objective

The objective of this project is to independently learn Node.js and Express and build a complete working website with backend functionality, then deploy it online. This project demonstrates full-stack development skills including REST API design, file handling, and database integration.

Method Route Description
