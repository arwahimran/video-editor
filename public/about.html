🎬 What is CutLab?

CutLab is a full-stack web-based video editing tool built with Node.js and Express. It allows users to create video editing projects, upload video files, trim clips by setting start and end times, add notes, and manage everything through a clean and fully responsive interface.

🎯 Project Objective

The objective of this project is to independently learn Node.js and Express and build a complete working website with backend functionality. This project demonstrates full-stack development skills including REST API design, file handling, database integration and responsive UI design.

🔌 API Documentation

All available REST API endpoints

Method Route Description
GET /api/projects Get all projects
GET /api/projects/:id Get single project
POST /api/projects Create new project
PUT /api/projects/:id Update a project
DELETE /api/projects/:id Delete a project
GET /api/projects/:id/clips Get clips for a project
POST /api/projects/:id/clips Add clip with video upload
DELETE /api/clips/:id Delete a clip
GET /api/health API health check
